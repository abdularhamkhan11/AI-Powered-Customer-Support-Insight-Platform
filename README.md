# AI-Powered Customer Support Insight Platform

## Overview
This project is an AI-based system that analyzes customer support tickets and generates useful business insights. It helps in understanding customer complaints, detecting sentiment, and identifying high-risk issues affecting revenue.

## Features
- Sentiment Analysis (Positive / Negative / Neutral)
- Automatic Ticket Categorization
- Detection of common customer issues
- High-risk revenue impact detection
- Business insights from support data

## How It Works
The system reads customer support ticket data from a CSV file, applies rule-based AI logic to detect sentiment and categories, and then generates insights such as complaint trends, sentiment distribution, and revenue risk analysis.

## Output
- Total tickets analyzed
- Sentiment distribution
- Top issue categories
- High-risk tickets (revenue impact)
- Processed dataset preview

## Tech Stack
- Python
- Pandas
- Rule-based NLP logic

## How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run the project:
python app.py
